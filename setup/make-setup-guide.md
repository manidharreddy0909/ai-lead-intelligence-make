## Make Setup Guide

### Prerequisites
- Make account
- Google account
- AI API key (OpenAI or Gemini)

### Steps
1. Create a Google Form with Name, Email, Company, and Message fields
2. Connect the form to Google Sheets
3. Create a Make scenario
4. Add Google Sheets "Watch New Rows" trigger
5. Add AI chat completion module
6. Parse AI JSON output
7. Add router with Hot/Warm/Cold conditions
8. Send emails using Gmail
9. Update the same row in Google Sheets
10. Turn the scenario ON

The system is now live.
